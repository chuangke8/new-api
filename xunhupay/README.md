# Skills 仓库说明

虎皮椒支付（xunhupay）API集成，支持发起支付、查询订单、退款、验证回调通知、签名生成。当用户需要对接虎皮椒支付、集成微信/支付宝个人收款接口、处理虎皮椒支付回调签名验证、生成支付二维码时使用。
## 仓库内容

- 各类 Skill 的定义文件（如 `SKILL.md`）
- 与 Skill 配套的脚本（如 `scripts/`）
- 多语言demo（如 `examples/`）

## 使用方式

1. 进入对应 Skill 目录，阅读 `SKILL.md` 了解功能和触发条件。
2. 触发词包括："虎皮椒支付"，"xunhupay"，"微信/支付宝个人收款"，"虎皮椒回调"。
3. 根据实际使用反馈持续更新规则、流程和文档。

## 适用场景

- 网站对接虎皮椒支付

欢迎使用虎皮椒支付 Skill，一起完善这个能力库。
